# Event Web Scrapping

Scrapping for some web pages, where multiple events are posted using BeautifulSoup module.

To start run following commands:

Install BeautifulSoup library:

`pip install beautifulsoup4`

`git clone https://github.com/vishwasluhana/event-web-scrapping`

`cd event-web-scrapping`

`python main.py`

It will print everything scrapped from the web pages stored in "data" directory.

Mainly scrapping all the event details.
